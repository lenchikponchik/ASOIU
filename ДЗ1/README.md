# ДЗ1. Расстояние Дамерау-Левенштейна

Консольное приложение на C# вычисляет расстояние редактирования между двумя строками. Поддерживаются операции вставки, удаления, замены и транспозиции соседних символов.

## Запуск

```bash
dotnet run --project EditDistance.csproj
```

Программа работает в цикле:

1. Пользователь вводит первую строку.
2. Пользователь вводит вторую строку.
3. Программа выводит расстояние.
4. Для выхода в качестве первой строки вводится `exit`.

## Пример

```text
Введите первую строку: пример
Введите вторую строку: примре
Расстояние: 1
```

Расстояние равно 1, потому что используется одна транспозиция соседних символов `е` и `р`.
